1. 數字遊戲#

解題步驟#

  1. 初始化變數: 宣告一個大小為 9 的整數陣列 cnt 用於計算 1 到 9 每個數字出現的次數，並將其全部初始化為 0。宣告一個整數 max_cnt 用於儲存「出現最多次的次數」，並初始化為 0。
  2. 讀取輸入與統計: 使用 for 迴圈執行 3 次。在迴圈中：
    • 讀取一個整數 P。
    • 將 cnt[P-1] 的值加 1（例如，讀到 6，就增加 cnt[5] 的計數）。
    • 檢查 cnt[P-1] 是否大於 max_cnt，如果是，則更新 max_cnt = cnt[P-1]。
  3. 輸出最大次數: 迴圈結束後，max_cnt 儲存的就是出現最多次的數字的次數（即 $P$），將其輸出。
  4. 輸出不重複數字: 使用 一個 for 迴圈，從 9 遞減到 1 (變數 i 從 9 到 1)。
    • 在迴圈中，檢查 cnt[i-1] 是否大於 0。
    • 如果大於 0，表示數字 i 在輸入中出現過，將 i 輸出。

#include<iostream>
using namespace std;

int main(){
    int P, max_cnt = 0;
    int cnt[9] = {0};
    for(int i=0;i<3;i++){
        cin >> P;
        cnt[P-1]++;
        if(cnt[P-1]>max_cnt){
            max_cnt = cnt[P-1];
        }
    }
    cout << max_cnt << ' ';
    for(int i=9;i>=1;i--){
        if(cnt[i-1]>0){
            cout << i << ' ';
        }
    }
}

2. 字串解碼#

解題步驟#

  1. 讀取輸入資料: 讀取加密層數 m、字串長度 n。接著使用 vector<string> 儲存 m 個加密鍵 keys，最後讀取經過 m 層加密後的字串 encode。
  2. 建立反向解密迴圈: 加密是從第 0 層進行到第 m-1 層。因此，解密必須反向操作，使用一個 for 迴圈從 i = m-1 迭代到 0。
  3. 反向解密 - 步驟 2 (重建字串): 在每一層解密中，我們需要反轉加密的步驟 2。
    • 加密是「從 S 的頭/尾取字元，放到 T 的尾端」。
    • 解密就是「從 encode (即 T) 的尾端取字元，根據 key 放回 decode (即 S) 的頭端或尾端」。
    • 我們從 j = n-1 迭代到 0 來反向讀取 key。
    • 如果 key[j] == '1' (加密時從 S 尾端取)，解密時就將 encode.back() 放到 decode 的 尾端。
    • 如果 key[j] == '0' (加密時從 S 前端取)，解密時就將 encode.back() 放到 decode 的 前端。
  4. 計算 ‘1’ 的數量: 在執行步驟 3 的同時，計算當前 key 中 ‘1’ 的總數 cnt。
  5. 反向解密 - 步驟 1 (交換字串): 檢查 cnt 是否為奇數。如果是，代表加密時曾執行過「交換字串」的操作。因此在解密時，我們需要執行 反向 的交換（將後半部移到前面，前半部移到後面）來還原字串。
  6. 更新字串狀態: 將當前層解密完成的 decode 字串，賦值回 encode 變數，作為下一層解密迴圈的輸入。
  7. 輸出最終結果: 當 m 層迴圈全部解密完畢後，encode 變數中儲存的即為原始字串，將其輸出。

#include<iostream>
#include<vector>

using namespace std;

int main(){
    int m,n;
    cin >> m >> n;
    string encode;
    vector<string> keys(m);
    for(int i=0;i<m;i++){
        cin >> keys[i];
    }
    cin >> encode;
    for(int i=m-1;i>=0;i--){
        string key = keys[i];
        int cnt = 0;
        string decode = "";
        for(int j=n-1;j>=0;j--){
            if(key[j]=='1'){
                cnt++;
                decode = decode + encode.back();
                encode.pop_back();
            }else{
                decode = encode.back() + decode;
                encode.pop_back();
            }
        }
        if(cnt%2==1){
            if(n%2==1){
                decode = decode.substr((n-1)/2+1,n)
                       + decode[(n-1)/2]
                       + decode.substr(0,(n-1)/2);
            }else{
                decode = decode.substr(n/2,n)
                       + decode.substr(0,n/2);
            }
        }
        encode = decode;
    }
    cout << encode;
}
