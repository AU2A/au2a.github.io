1. 巴士站牌#

解題步驟#

  1. 讀取輸入並初始化: 讀取站牌總數 n 和第一個巴士站的座標 (px, py)。
  2. 初始化最大/最小值: 初始化最大距離 maxn 為 0（或一個極小值），最小距離 minn 為一個極大值（例如 1000，根據題目範圍 400 已經足夠）。
  3. 迴圈處理剩餘站牌: 使用 for 迴圈，從 $i=1$ 迭代到 $n-1$，處理剩下的 $n-1$ 個巴士站（即從第 2 個站到第 $n$ 個站）。
  4. 讀取並計算距離: 在迴圈中，讀取下一個巴士站的座標 (x, y)。接著計算目前站 (x, y) 與前一個站 (px, py) 之間的曼哈頓距離 d。
  5. 更新最大值和最小值: 比較剛算出的距離 d 與 maxn 和 minn。
    • 如果 d 大於 maxn，則更新 maxn = d。
    • 如果 d 小於 minn，則更新 minn = d。
  6. 更新前一站位置: 在迴圈結束前，將目前的座標 (x, y) 存為 “前一個” 座標 (px, py)，供下次迴圈計算距離時使用。
  7. 輸出結果: 迴圈結束後，maxn 和 minn 即為所求的最大值和最小值，將它們依序輸出。

#include<iostream>

int main(){
    int n, x, y, px, py, maxn = 0, minn = 1000, d;
    // 讀取總站數 n 和第一個站的座標
    cin >> n >> px >> py;
   
    // 迴圈處理第 2 個到第 n 個站
    for(int i=1;i<n;i++){
        // 讀取下一個站的座標
        cin >> x >> y;
       
        // 計算與前一站的曼哈頓距離
        d = abs(px - x) + abs(py - y);
       
        // 更新最大距離
        if(maxn < d){
            maxn = d;
        }
       
        // 更新最小距離
        if(minn > d){
            minn = d;
        }
       
        // 將目前座標設為 '前一個' 座標，供下次迴圈使用
        px = x;
        py = y;
    }
   
    // 輸出最大值和最小值
    cout << maxn << ' ' << minn;
}

1. 跳柵遊戲#

解題步驟#

  1. 讀取初始生命值和陷阱條件: 讀取初始生命值 k，以及兩個陷阱的條件 x1、y1 和 x2、y2。
  2. 初始化位置和生命值: 初始化位置 p 為 0，生命值為 k。
  3. 模擬遊戲過程: 使用 while 迴圈模擬遊戲過程，直到生命值小於等於 0。
  4. 更新位置和生命值: 在每次迴圈中，根據當前生命值更新位置 p，並根據 p 是否為 x1 或 x2 的倍數計算生命值的損失。
  5. 輸出最終位置: 當生命值小於等於 0 時，輸出當前位置 p。

int main(){
    int k, x1, x2, y1, y2;
    cin >> k;
    cin >> x1 >> y1;
    cin >> x2 >> y2;
    int p = 0;
    while(k>0){
        p += k;
        int dmg = 0;
        if(p % x1 == 0){
            dmg += y1;
        }
        if(p % x2 == 0){
            dmg += y2;
        }
        k -= dmg;
    }
    cout << p;
}

2. 運貨站#

解題步驟#

  1. 讀取地圖資訊與初始化: 讀取倉庫的列數 、行數 以及貨物數量 。建立一個二維陣列 arr[R][C+4] 來模擬倉庫。為了方便判斷「推到不能前進為止」，我們在最左側（第 0 欄）預填「障礙物」作為牆壁。
  2. 模擬貨物推入: 依序讀取每個貨物的類型 與離頂部距離 。貨物會從倉庫右側進入，並向左滑動直到碰撞到現有的貨物或左側牆壁。
  3. 碰撞偵測與放置:
  • 根據不同形狀（A 到 E），定義其佔用的相對座標。
  • 由右向左（i 從 C+1 遞減到 0）檢查貨物的「左邊緣」是否會碰撞到已存在的貨物。
  • 若找到碰撞點，則嘗試將貨物放在碰撞點的右側一格。
  1. 檢查邊界（丟棄機制）: 判斷放置後的貨物右邊緣是否超出了倉庫的寬度 。若超出，則該貨物會被丟棄（out++），且不改變倉庫狀態。
  2. 統計結果: 模擬結束後，遍歷倉庫陣列，統計數值仍為 0 的空格數量，並輸出空格數與被丟棄的貨物總數。

貨物形狀說明#

根據程式碼邏輯，這五種貨物的佔據方式如下：

  • A: 垂直長條 (1x4)。
  • B: 水平長條 (3x1)。
  • C: 方塊 (2x2)。
  • D: L 型貨物。
  • E: 凸型或 T 型貨物。

#include<iostream>
using namespace std;

int main(){
    int r, c, n, out = 0, y;
    char t;
    if (!(cin >> r >> c >> n)) return 0;

    // 建立倉庫，多開空間避免邊界檢查溢位
    int arr[35][60] = {0};

    // 設定左側邊界牆壁
    for(int i = 0; i < r; i++){
        arr[i][0] = 1;
    }

    for(int nn = 0; nn < n; nn++){
        cin >> t >> y;
        if(t == 'A'){
            for(int i = c + 1; i >= 0; i--){
                if(arr[y][i] || arr[y+1][i] || arr[y+2][i] || arr[y+3][i]){
                    if(i + 1 <= c){
                        arr[y][i+1] = arr[y+1][i+1] = arr[y+2][i+1] = arr[y+3][i+1] = 1;
                    }else{ out++; }
                    break;
                }
            }
        }else if(t == 'B'){
            for(int i = c + 1; i >= 0; i--){
                if(arr[y][i]){
                    if(i + 3 <= c){
                        arr[y][i+1] = arr[y][i+2] = arr[y][i+3] = 1;
                    }else{ out++; }
                    break;
                }
            }
        }else if(t == 'C'){
            for(int i = c + 1; i >= 0; i--){
                if(arr[y][i] || arr[y+1][i]){
                    if(i + 2 <= c){
                        arr[y][i+1] = arr[y][i+2] = arr[y+1][i+1] = arr[y+1][i+2] = 1;
                    }else{ out++; }
                    break;
                }
            }
        }else if(t == 'D'){
            for(int i = c + 1; i >= 0; i--){
                if(arr[y+1][i] || arr[y][i+2]){
                    if(i + 3 <= c){
                        arr[y+1][i+1] = arr[y+1][i+2] = arr[y+1][i+3] = arr[y][i+3] = 1;
                    }else{ out++; }
                    break;
                }
            }
        }else if(t == 'E'){
            for(int i = c + 1; i >= 0; i--){
                if(arr[y][i+1] || arr[y+1][i] || arr[y+2][i]){
                    if(i + 2 <= c){
                        arr[y][i+2] = arr[y+1][i+1] = arr[y+1][i+2] = arr[y+2][i+1] = arr[y+2][i+2] = 1;
                    }else{ out++; }
                    break;
                }
            }
        }
    }

    int blank = 0;
    for (int i = 0; i < r; i++){
        for(int j = 1; j <= c; j++){
            if(arr[i][j] == 0) blank++;
        }
    }
    cout << blank << " " << out << endl;
    return 0;
}
