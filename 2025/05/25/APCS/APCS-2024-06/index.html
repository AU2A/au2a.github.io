1. 特技表演

解題步驟

  1. 讀取輸入資料: 讀取大樓的總數 n，接著使用一個 for 迴圈讀取 n 棟大樓分別的高度，並將它們儲存在一個陣列 h 中。
  2. 初始化變數: 初始化兩個整數變數。max_d 用來儲存找到的「最長」滑翔路徑長度，初始為 0。d 用來儲存「當前」正在計算的連續滑翔路徑長度，初始為 1（因為任何一棟大樓本身都構成長度為 1 的路徑）。
  3. 遍歷大樓陣列: 使用 for 迴圈從第一棟大樓 (i = 0) 檢查到倒數第二棟大樓 (i < n-1)。
  4. 判斷路徑是否延伸: 在迴圈中，比較目前大樓 h[i] 和下一棟大樓 h[i+1] 的高度。
    • 如果 h[i] > h[i+1]，表示符合「越來越低」的條件，滑翔路徑可以繼續，因此將當前路徑長度 d 增加 1 (d++)。
    • 如果 h[i] <= h[i+1]，表示路徑中斷（高度沒有變低），因此重置當前路徑長度 d 為 1（從 h[i+1] 重新開始計算）。
  5. 更新最長路徑: 在每次判斷（或重置）d 之後，都要檢查 d 是否大於 max_d。如果 d 更大，就更新 max_d = d。
  6. 輸出結果: 迴圈結束後，max_d 中儲存的就是整個過程中最長的路徑長度，將其輸出。

分段步驟程式碼

讀取輸入資料

這段程式碼首先讀取大樓的數量 n，然後宣告一個大小為 n 的陣列 h，並使用迴圈讀取每棟大樓的高度。

初始化變數

這段程式碼初始化 max_d 為 0（最終的最長路徑）和 d 為 1（當前計算的路徑）。

遍歷、比較並更新最長路徑

這段程式碼是解題的核心。它遍歷 n-1 次（比較 h[0] 和 h[1]、h[1] 和 h[2]…）。

  • 如果 h[i] > h[i+1]，當前路徑 d 增加。
  • 否則，路徑中斷，d 重置為 1。
  • 每一次迴圈都會檢查 d 是否是新的最大值 max_d。

輸出結果

這段程式碼在所有比較完成後，輸出儲存的最長路徑 max_d。

2. 電子畫布

解題步驟

  1. 引入函式庫與宣告: 引入 iostream (用於輸入輸出) 和 vector (用於建立二維畫布)。
  2. 宣告輔助函式: 宣告一個計算曼哈頓距離的函式 m_dis 的原型。
  3. 讀取基本資訊: 在 main 函式中，讀取畫布高度 h、寬度 w 和操作次數 n。
  4. 初始化畫布: 使用 vector<vector<int>> 建立一個 h x w 的二維向量 board，並將所有初始值設為 0。
  5. 模擬操作: 使用一個 for 迴圈執行 n 次：
    • 讀取該次操作的座標 (r, c)、範圍 t 和顏色值 x。
    • 使用 巢狀迴圈 遍歷畫布上的 每一個 座標 (hh, ww) (從 0 到 h-1，0 到 w-1)。
    • 呼叫 m_dis(hh, ww, r, c) 計算當前座標與操作中心 (r, c) 的曼哈頓距離。
    • 如果距離小於等於 t，則將 board[hh][ww] 的值加上 x。
  6. 輸出結果: n 次操作都完成後，使用巢狀迴圈遍歷 board，並依序印出每個格子的最終數值。
  7. 實作輔助函式: 在 main 函式之後，實作 m_dis 函式，計算並回傳 |hh - r| + |ww - c|。

分段步驟程式碼

引入函式庫與函式原型

這段程式碼引入必要的函式庫，並宣告 m_dis 函式，以便 main 函式可以呼叫它。

讀取輸入與初始化畫布

這段程式碼讀取畫布尺寸和操作次數，並動態建立一個 h x w 的二維向量 board。

模擬 N 次畫筆操作

這段程式碼是解題的核心。它使用一個迴圈來處理 n 次操作。對於每次操作，它都會遍歷 整個 畫布，檢查每個格子是否在影響範圍內，並如果是，就累加顏色值。

輸出畫布狀態

這段程式碼在所有操作完成後，依序印出二維向量 board 中的所有值。

曼哈頓距離函式實作

這段程式碼實作了計算曼哈頓距離的輔助函式。它計算兩個座標 (hh, ww) 和 (r, c) 之間 x 座標差的絕對值與 y 座標差的絕對值之和。
