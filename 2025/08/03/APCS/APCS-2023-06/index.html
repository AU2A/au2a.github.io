1. 路徑偵測

解題步驟

  1. 讀取輸入: 讀取座標點的總數 n。
  2. 初始化狀態:
    • 定義 d 來表示目前方向 (1=右, 2=上, 3=左, 4=下)。
    • 根據題目，初始方向為向右，所以 d = 1。
    • 初始化左轉 l、右轉 r、迴轉 re 的計數器為 0。
    • 讀取第一個點的座標 (x, y)，這將作為我們路徑的起始位置。
  3. 迴圈處理路徑: 使用 for 迴圈，從 i=1 到 n-1，讀取剩下的 n-1 個點。
  4. 讀取下一個點: 在迴圈中，讀取下一個點的座標 (nx, ny)。
  5. 判斷轉向:
    • 使用 if-else if 結構，根據目前方向 d 進行判斷。
    • 在
      d 的條件下，比較目前座標 (x, y) 和下一個座標 (nx, ny) 的相對位置。
    • 例如：如果 d=1 (目前向右)，而 ny > y (下一個點在上方)，這代表一次左轉。此時 l++ 並更新 d=2 (新方向為上)。
    • 如果 d=1 (目前向右)，而 nx < x (下一個點在左方)，這代表一次迴轉。此時 re++ 並更新 d=3 (新方向為左)。
    • 依此類推，完成所有四個方向的判斷。
  6. 更新目前位置: 在判斷完轉向後，將 x = nx 和 y = ny，使下一個點成為新的目前位置，以供下一次迴圈判斷。
  7. 輸出結果: 迴圈結束後，輸出 l, r, re 的值。

分段步驟程式碼

初始化狀態並讀取第一個點

這段程式碼讀取點的總數 n。初始化目前方向 d=1 (向右)，左轉 l、右轉 r、迴轉 re 計數器為 0。讀取第一個點的座標 x, y，這也確立了我們的初始位置。

迴圈處理所有轉向

這段程式碼是解題的核心。它遍歷 n-1 次，每次讀取下一個點 (nx, ny)。
接著，它檢查目前的方向 d：

  • d=1 (向右): 檢查 Y 軸變化（ny > y 左轉, ny < y 右轉）或 X 軸變化（nx < x 迴轉）。
  • d=2 (向上): 檢查 X 軸變化（nx > x 右轉, nx < x 左轉）或 Y 軸變化（ny < y 迴轉）。
  • d=3 (向左): 檢查 Y 軸變化（ny > y 右轉, ny < y 左轉）或 X 軸變化（nx > x 迴轉）。
  • d=4 (向下): 檢查 X 軸變化（nx > x 左轉, nx < x 右轉）或 Y 軸變化（ny > y 迴轉）。

每次偵測到轉向，就增加對應的計數器 (l, r, re) 並更新方向 d。如果沒有轉向（例如 d=1 且 nx > x），則 d 保持不變。
最後，將 x 和 y 更新為 nx 和 ny，為下一次迴圈做準備。

輸出結果

迴圈結束後，l, r, re 已記錄所有轉向次數，將它們依序輸出。

2. 特殊位置

解題步驟

  1. 讀取矩陣維度和內容: 讀取 $n$、$m$ 以及 $n \times m$ 矩陣 a 的所有元素。
  2. 初始化答案儲存: 宣告一個 vector<pair<int, int>> ans 來儲存所有特殊位置的座標。
  3. 遍歷所有位置: 使用兩層 for 迴圈，依序檢查矩陣中的每一個位置 (i, j)。
  4. 計算範圍內總和: 對於當前位置 (i, j)，令 x = a[i][j]。接著使用兩層 for 迴圈遍歷矩陣中所有位置 (x_pos, y_pos)。
  5. 檢查曼哈頓距離: 計算 (i, j) 與 (x_pos, y_pos) 的曼哈頓距離 |i - x_pos| + |j - y_pos|。
  6. 累加總和: 如果距離小於等於 x (即 a[i][j])，則將 a[x_pos][y_pos] 加入總和 sum。
  7. 檢查特殊位置條件: 對於 (i, j)，計算完 sum 後，檢查 sum % 10 是否等於 x (即 a[i][j])。如果相等，則將 (i, j) 存入 ans 中。
  8. 輸出結果: 遍歷結束後，首先輸出 ans 的大小 (即特殊位置的總數)，然後依序（字典序）輸出 ans 中儲存的每一個座標。

分段步驟程式碼

讀取矩陣維度和內容

這段程式碼讀取矩陣的維度 $n$ 和 $m$，並使用兩層迴圈讀取 $n \times m$ 個矩陣元素，存儲在 a 陣列中。

初始化答案儲存

這段程式碼初始化一個 vector，其元素型別為 pair<int, int>，用來儲存所有找到的特殊位置座標。

遍歷、計算並檢查特殊位置

這是解題的核心。

  1. 最外層的兩層迴圈 (i, j) 用於遍歷矩陣中的每一個位置。
  2. 對於每個 (i, j)，初始化 sum = 0。
  3. 內層的兩層迴圈 (x, y) 用於遍歷所有矩陣位置，以計算曼哈頓距離。
  4. if 判斷式檢查 (x, y) 是否在 (i, j) 的曼哈頓距離 a[i][j] 範圍內。
  5. 如果在範圍內，將 a[x][y] 累加到 sum。
  6. 內層迴圈結束後，檢查 sum % 10 是否等於 a[i][j]。如果成立，表示 (i, j) 是一個特殊位置，將其座標 (i, j) 加入 ans 中。

輸出最終答案

這段程式碼首先輸出 ans.size()，即特殊位置的總數。接著，使用一個迴圈遍歷 ans，並依序輸出儲存的每個座標。由於 ans 是按照 (i, j) 字典序（先行後列）的順序加入的，因此輸出時自然會符合字典序。
