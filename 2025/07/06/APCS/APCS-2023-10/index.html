1. 機械鼠

解題步驟

  1. 讀取初始位置和食物數量：讀取老鼠的初始位置 x 和食物的總數 n。
  2. 初始化變數：
    • min：用來儲存最左邊食物的位置，初始為一個極大值 (例如 101)。
    • max：用來儲存最右邊食物的位置，初始為一個極小值 (例如 -101)。
    • step：用來計算老鼠左邊的食物數量，初始為 0。
    • now：用來暫存當前讀取的食物位置。
  3. 遍歷所有食物：使用 for 迴圈讀取 n 個食物的位置。
  4. 分類食物位置：
    • 在迴圈中，檢查每個食物的位置 now。
    • 如果 now < x，表示食物在老鼠左邊。將左邊食物計數 step 加 1，並更新 min (如果 now 比當前的 min 更小)。
    • 如果 now > x，表示食物在老鼠右邊。更新 max (如果 now 比當前的 max 更大)。
  5. 計算右側食物數量：迴圈結束後，右側食物的數量即為 n - step。
  6. 比較並輸出：
    • 比較左側食物數量 step 和右側食物數量 n - step。
    • 如果 step 比較大，表示往左走能吃到更多食物。輸出 step 和最左邊的位置 min。
    • 否則 (右側數量大於或等於左側數量)，表示往右走能吃到更多或相等數量的食物。輸出 n - step 和最右邊的位置 max。

分段步驟程式碼

讀取輸入並初始化變數

這段程式碼讀取老鼠的初始位置 x 和食物數量 n。接著初始化四個變數：

  • min 設為 101 (大於最大可能位置 100)，用來找最小值。
  • max 設為 -101 (小於最小可能位置 -100)，用來找最大值。
  • step 設為 0，用來計算左側食物的數量。
  • now 用於在迴圈中儲存當前讀取的食物位置。

遍歷並分類食物

這段程式碼使用 for 迴圈讀取 n 個食物的位置。

  • 如果當前食物位置 now 小於 x，表示食物在老鼠左側。step (左側計數) 加 1，並檢查 now 是否比當前的 min 更小，是的話就更新 min。
  • 否則 (食物在右側)，檢查 now 是否比當前的 max 更大，是的話就更新 max。

比較並輸出結果

這段程式碼比較左側食物的數量 step 和右側食物的數量 n - step。

  • 如果 step (左側) 比較多，就輸出左側的數量 step 和最左側的位置 min。
  • 如果 n - step (右側) 比較多或兩者相等，就輸出右側的數量 n - step 和最右側的位置 max。

2. 卡牌遊戲

解題步驟

  1. 讀取表格大小和內容: 讀取 $n$、$m$ 以及 $n \times m$ 表格的所有數字。
  2. 初始化分數: 初始化總分 score = 0。
  3. 模擬消除過程: 使用一個 while 迴圈來模擬消除過程。這個迴圈會一直執行，直到某一輪「沒有任何一對牌」被成功消除為止。
  4. 標記本輪狀態: 在 while 迴圈的開頭，設定一個布林變數 chk = false，用來追蹤這一輪是否發生了消除。
  5. 遍歷尋找配對:
    • 使用兩層 for 迴圈遍歷表格中的每個格子 (i, j)。
    • 如果格子 (i, j) 尚未被消除 (我們用 $\geq 0$ 的值表示尚未消除)，則開始尋找配對。
  6. 檢查垂直方向:
    • 從 (i, j) 向下 (即 r 從 i+1 到 n-1) 檢查同一行 j 上的格子 (r, j)。
    • 如果找到 num[i][j] == num[r][j]，表示找到配對。此時：
      1. 將 num[i][j] 的值加入 score。
      2. 將 num[i][j] 和 num[r][j] 都設為 -1 (標記為已消除)。
      3. 設定 chk = true (表示本輪有成功消除)。
      4. break 停止垂直搜尋。
    • 如果在找到配對前，先遇到一個 num[r][j] >= 0 (即一個尚未消除的 不同 數字)，表示路徑被阻擋，break 停止垂直搜尋。
  7. 檢查水平方向:
    • (在檢查水平前，再次檢查 num[i][j] 是否 $\geq 0$，因為它可能剛好在垂直檢查中被消除了)。
    • 從 (i, j) 向右 (即 r 從 j+1 到 m-1) 檢查同一列 i 上的格子 (i, r)。
    • 套用與垂直檢查相同的邏輯：找到配對就加分、標記為 -1、設 chk = true 並 break；遇到障礙物也 break。
  8. 輸出最終分數: 當 while 迴圈結束 (因為 chk 保持為 false) 時，表示所有可消除的配對都已處理完畢，輸出總分 score。

分段步驟程式碼

讀取輸入與初始化

這段程式碼讀取表格的行數 n 和列數 m。接著，宣告一個 $n \times m$ 的二維陣列 num 來儲存表格內容，並初始化分數 score 為 0。最後，使用巢狀迴圈讀取所有格子的數值。

模擬消除迴圈

這段程式碼是模擬的核心。while(chk) 迴圈會持續執行，只要上一輪 (chk = true) 有成功消除。每輪開始時，先將 chk 重設為 false。接著，使用兩層 for 迴圈遍歷表格中的每一個格子 (i, j)。

檢查垂直與水平配對

這段程式碼位於 for(j...) 迴圈內部。

  1. 首先檢查 (i, j) 是否已被消除 (值 < 0)，若是則 continue。
  2. 垂直檢查: 向下搜尋 (r, j)。如果找到相同的值，則加分、將兩者都設為 -1、將 chk 設為 true，並 break。如果先找到 $\geq 0$ 的不同值 (障礙物)，也 break。
  3. 水平檢查: 再次檢查 (i, j) (因為它可能剛被垂直檢查消除)。如果未被消除，則向右搜尋 (i, r)，執行與垂直檢查相同的邏輯。

輸出最終分數

當 while(chk) 迴圈結束 (即 chk 在一整輪中都保持為 false)，表示沒有更多可以消除的配對。此時，輸出累計的總分 score。
