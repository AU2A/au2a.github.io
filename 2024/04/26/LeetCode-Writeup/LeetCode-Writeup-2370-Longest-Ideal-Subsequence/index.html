Longest Ideal Subsequence

這題是個基礎的 DP 題，很值得練習。

Idea 1 TLE

第一種方法，就是當前的字往前檢查可以接的字有哪些，選最高的。

Main function

第 n 個字時，會檢查 n-1 到 1 的字字串的長度

先檢查當前的字和前面的字距離有沒有小於等於 k，如果有才符合題目條件。

有的話再檢查是否大於目前記錄到的長度。

找到前面可以接續的最長長度後，加上自己把值存進自己位置的陣列，讓下一個字往前檢查時可以讀到。

題目要求回傳最長的符合題意的字串長度，因此用個變數順便紀錄，這樣最後直接回傳即可。

Solution 1

第一個方法好理解，但問題是每次往前檢查不知道要檢查到第幾個，因此都檢查到第一個，執行時間是 O(n^2)，雖然有用到 DP 的概念，但不怎麼有效率。

如何有效的紀錄每個字的狀態？

第二種方法，不同於紀錄字串每個位置的長度，而是用每個字母紀錄最長的長度。

因為字元差需要在 k 之間，因此只需檢查當前字母+-k 的字母就可以，

舉例像是字母 c，k=1，那只需檢查 abcdefgh 就行。

檢查後將新的值放到 c 的位置，讓下一個字可以繼續更新。

Main function

每個字都還是要跑一遍，用nowWordIdx將當前的字轉換成數字 0 ~ 25

透過nowWordIdx，我僅需檢查 +-k 的字，但要注意nowWordIdx-k需大於 0，nowWordIdx+k需小於 25，避免超出範圍。

當前的字找到最長的值後，加上自己後存在自己字母索引的地方，讓下一個字檢查時可以更新。

題目要求回傳最長的符合題意的字串長度，因此用個變數順便紀錄，這樣最後直接回傳即可。
