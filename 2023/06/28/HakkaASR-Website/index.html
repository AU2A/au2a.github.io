Docker 執行

使用 Docker 時，可以使用以下指令：

psw為密碼，需要密碼才能執行，如需要請向我詢問。
domain為網址，設定網頁使用的網址
port1為外部的網頁 port
port2為外部的解碼用 port

-p 443:443網頁使用的是https通道
-p 5002:5002網頁解碼使用的通道

執行

全部執行

如果要在本地端執行，請執行test.sh

分開執行

請在專案目錄下執行檔案，不然部分指令相對路徑會錯誤

網頁執行

執行網頁前，需先安裝網頁需要的套件
請先到website/底下安裝

在回到專案目錄執行網頁

whisper 解碼執行

安全憑證

因為網頁有使用的錄音功能，所以網頁需要走https協定，因此需要申請安全憑證。
我是使用LetsEncrypt的安全憑證，每 90 天需要重新申請一次。
參考網站
請將申請後的安全憑證檔案放到keys/底下即可運作網頁。
檔案有三：cert.pem, chain.pem, privkey.pem
